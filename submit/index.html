Controls:

Movement: w-s, a-d, q-e (a and d are swapped from the shell)
Looking: SHIFT + [w-s, a-d, q-e] (ditto)

Select Mesh: arrow keys (up-down scrolls ellipsoids, right-left scrolls other meshes)
Manipulate Mesh: o-l, k-;, i-p; (hold SHIFT as well to rotate)

Toggle texture blending modes: b
- 0 : blend correctly
- 1 : only texture color
- 2 : only lighting color
- 3 : z-buffer as color

Toggle transparency: v
